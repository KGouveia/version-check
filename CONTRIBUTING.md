# Contributing

Software Version Tracker is a **closed project**. The repository is published for reference and personal use, but **this project is not accepting pull requests, feature requests, bug reports, issues, or other contributions**.

There is no contribution workflow. The project is maintained for the author’s own use.

## What you can do instead

- **Fork** the repository under the MIT license (see `package.json`) and maintain your own copy if you want to change it.
- **Run from source** for personal use — see [README.md](README.md) for installation and usage instructions.

Forks and local changes are your responsibility; the maintainer does not provide support for derivative work.
